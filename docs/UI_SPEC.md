# 통합 UI 명세서 (UI & Design Specification)

본 문서는 Smart Portfolio 프로젝트의 사이트 맵, 통합 디자인 가이드라인, 그리고 화면별 UI 상세 명세를 하나로 묶어 정리한 통합 문서입니다.

---

## 🗺️ 1. 사이트맵 및 네비게이션 (Sitemap)

**공통 레이아웃 요소**
- **상단 네비게이션 바 (Header / GNB)** : 로고(메인 이동), 메뉴 탭(`Home`, `Project`, `Resume`, `Admin`)
- **플로팅 AI 챗봇 위젯** : 우측 하단에 항상 떠 있는 채팅 버튼 (단, 관리자 페이지에서는 숨김 처리)
- **공통 푸터 (Footer)** : 저작권 정보 및 SNS/GitHub 링크

**사이트맵 트리 구조**
```text
🏠 메인 페이지 ( / )
 │
 ├── 📂 프로젝트 목록 ( /project )
 │    └── 📝 프로젝트 상세 ( /project/[id] )
 │
 ├── 📄 이력 및 경험 ( /resume )
 │
 └── 💬 AI 챗봇 전용 화면 ( /chat ) - 모바일 등에서 집중할 수 있는 풀스크린 챗봇
 
⚙️ 관리자 권한 영역 (Admin)
 │
 └── 🔒 관리자 로그인 ( /admin/login )
      │
      └── 📊 관리자 대시보드 ( /admin )
           ├── 👤 프로필 관리
           ├── 📂 프로젝트 관리
           └── 🔄 챗봇 데이터 설정
```

## 🎨 2. 통합 디자인 가이드 (Design Guide)

### 🎯 디자인 컨셉
- **키워드**: 깔끔함(Clean), 미니멀리즘(Minimal), 전문성(Professional), 따뜻함(Warmth)
- **테마**: 다크 모드 (Dark Mode Default)

### 🖌️ 주요 색상 팔레트
| 용도 | HEX 코드 | 설명 |
| :--- | :--- | :--- |
| **Background** | `#18181B` | 화면 전체 배경 (따뜻한 다크 징크) |
| **Surface** | `#27272A` | 카드, 모달, 패널 등의 표면 배경색 |
| **Primary** | `#E5A970` | 주요 버튼, 강조 텍스트용 따뜻한 베이지-오렌지(Soft Amber) |
| **Secondary** | `#3F3F46` | 보조 버튼, 구분선(Border) 색상 |
| **Accent** | `#E07A5F` | 알림 뱃지, 에러, 강조 포인트 (Soft Coral) |
| **Text** | `#F4F4F5` | 기본 텍스트 (눈부심 방지 Off White) |
| **Text Muted** | `#A1A1AA` | 보조 텍스트 (Dimmed Gray) |

### ✍️ 타이포그래피 (Typography)
- **제목용 (Headings)** : `Outfit` (영문) / `Pretendard` (한글)
- **본문용 (Body)** : `Pretendard` (자간 `-0.02em`, 줄간격 `1.6`)

### 📐 여백 및 간격 기준 (Grid System)
- **미세 간격** : `4px`, `8px`
- **요소 간격** : `16px`, `24px`
- **섹션 내부 패딩** : `32px`, `48px`
- **섹션 간 여백** : `80px` ~ `120px`

### 🔲 UI 요소 스타일링
- **모서리 둥글기 (Border Radius)**: 입력필드/버튼 `8px`, 프로젝트 카드 `16px`, 뱃지 `999px`, 원형이미지 `50%`
- **그림자 (Shadows)**: 
  - 기본 상태: `box-shadow: 0 4px 20px rgba(0, 0, 0, 0.4)`
  - 호버 상태: `box-shadow: 0 8px 30px rgba(0, 0, 0, 0.6); transform: translateY(-4px)`
- **전환 효과**: `transition: all 0.2s ease-in-out`

### 📱 반응형 디자인 기준 (Breakpoints)
- **모바일 (`0~768px`)**: 1열 레이아웃, 상단 햄버거 메뉴, 챗봇 활성화 시 전체 화면
- **태블릿 (`769~1024px`)**: 2열 그리드 배치, 축소된(Collapsed) 관리자 사이드바
- **데스크톱 (`1025px~`)**: 3열 그리드 배치, 텍스트 포함 풀사이즈 네비게이션

## 🖥️ 3. 화면별 UI 상세 명세 (Screen Specs)

### 메인 페이지 (/)
- **상단** : 네비게이션바 (로고 왼쪽, 메뉴 오른쪽)
- **히어로 섹션** : 프로필 사진(200*200px 원형) + 이름 + 한 줄 소개 + CTA 버튼("프로젝트 보러가기")
- **스킬 섹션** : 둥근 모서리 뱃지 태그 (배경색 없음, 텍스트 아웃라인 강조)
- **경력 섹션** : 타임라인 형태 (연도/회사명/역할/설명)
- **반응형 동작** : 데스크톱은 좌우 분할, 모바일은 1열 세로 배치

### 프로젝트 목록 페이지 (/project)
- **필터 영역** : 상단에 카테고리별 필터 버튼 배치 (전체/웹/앱/디자인 등)
- **리스트 영역** : 썸네일 + 제목 + 카테고리 태그 + 설명으로 구성된 카드 그리드
- **상호작용** : 카드 호버 시 살짝 확대(Scale) 및 짙은 그림자. 클릭 시 상세 페이지(`/project/[id]`) 이동

### 프로젝트 상세 페이지 (/project/[id])
- **히어로 영역** : 프로젝트 대표 대형 썸네일 또는 스크린샷 갤러리 슬라이더
- **상세 콘텐츠** : 프로젝트 상세 마크다운 설명, 사용 기술 스택 나열, 주요 관련 링크 버튼(GitHub, 데모)
- **하단 영역** : 이전/다음 프로젝트 네비게이션 

### AI 챗봇 위젯 (공통 컴포넌트)
- **닫힌 상태** : 우측 하단에 고정(Fixed)된 플로팅 원형 버튼 (60*60px)
- **열린 상태** : 400*500px 직사각형 패널 창. 상단(헤더 닫기) + 중앙(채팅 내역) + 하단(메시지 입력) 구조
- **상태 처리** : 웰컴 메시지 기본 출력, 답변 대기 중 타이핑 애니메이션, 관련 없는 질문에 대한 정중한 에러 피드백
- **모바일 대응** : 클릭 시 창 전체를 덮는 100% 풀스크린 뷰로 전환

### 관리자 로그인 화면 (/admin/login)
- **구조** : 화면 중앙에 최대 400px 너비로 정렬된 모던한 로그인 폼 패널
- **인증 제어** : 시스템에 설정된 특정 관리자 이메일로만 진입 허용, 실패 시 "권한 없음" 토스트 메시지

### 관리자 대시보드 화면 (/admin)
- **구조** : 좌측 메뉴 네비게이션(사이드바) + 우측 메인 콘텐츠 뷰 영역
- **주요 기능(탭)** :
  1. 프로필 관리 : 사진, 이력, 한 줄 소개 수정 폼 (하단 저장버튼 고정)
  2. 프로젝트 관리 : 기존 프로젝트 테이블 노출. 우측의 추가/수정/삭제 액션
  3. 데이터 모달창 : 프로젝트 정보(제목, 설명, 썸네일, 링크 등) 추가 입력 팝업
  4. 챗봇 지식 설정 : AI가 답변할 때 기준이 되는 데이터(텍스트/URL) 추가 폼
- **반응형 동작** : 모바일 진입 시 사이드바가 햄버거 형태의 드로어(Drawer) 슬라이드로 숨김 처리됨
