# 사이트맵 및 네비게이션 흐름 (Sitemap)

> 본 문서는 smart-portfolio 웹 서비스의 화면 간 이동 흐름과 계층 구조를 트리(Tree) 형태로 시각화한 문서입니다.

## 📌 공통 레이아웃 요소 (Common Components)
*모든 페이지(또는 특정 영역)에서 공통적으로 접근 가능한 UI 요소입니다.*

- **상단 네비게이션 바 (Header / GNB)**
  - 로고 (클릭 시 메인 뷰 `/` 로 이동)
  - 메뉴 탭: `Home`, `Resume`, `Admin`
- **플로팅 AI 챗봇 위젯 (Floating Chatbot)**
  - 우측 하단에 항상 떠 있는 버튼. 클릭 시 팝업 형태로 AI 채팅창 활성화. (단, `/chat` 전용 화면 및 관리자 페이지에서는 숨김 처리 가능)
- **공통 푸터 (Footer)**
  - 저작권 정보 및 이메일/SNS 링크 (GitHub, LinkedIn 등)

---

## 🗺️ 사이트맵 트리 구조 (Tree Structure)

```text
🏠 메인 프로필 ( / )
 │
 ├── 📂 프로젝트 상세 ( /project/[id] )
 │    └── (뒤로가기 또는 GNB를 통해 메인으로 복귀 가능)
 │
 ├── 📄 이력 및 경험 ( /resume )
 │
 └── 💬 AI 챗봇 전용 화면 ( /chat ) 
      └── 모바일 환경 등에서 챗봇에 집중하기 위해 위젯 대신 풀스크린으로 접근


⚙️ 관리자 권한 영역 (Admin)
 │
 └── 🔒 관리자 로그인 ( /admin/login )
      │
      └── (이메일 인증 성공 시 진입)
           │
           └── 📊 관리자 대시보드 ( /admin )
                │
                ├── 📝 대화 로그 상세 ( /admin/logs )
                │
                └── 🔄 포트폴리오 지식 설정 ( /admin/setup )
```
